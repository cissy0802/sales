# Topics Roadmap — 销售（Sales）

> **⚠️ routine 须知**：本表即全部计划。只写下面**已列出**的条目；**全部写完就停**——别自己发明新主题、别往本文件加行。写完只发一条 PushNotification 请 BigCat 补充。（新条目由 BigCat / deep-research 反哺加进本表，届时你自然继续写。）

**定位**：怎么把**价值 / 自己**卖出去并促成交换——理解需求 → 建立信任 → 讲清价值 → 处理异议 → 成交 → 长期关系。覆盖 B2B/B2C、创业者卖产品、个人卖自己（求职 / 融资 / 个人品牌）、日常说服的**落地打法**。分工：psychology 讲说服的**心理机制**、leadership 讲**对内影响与谈判**、investing 讲**钱**、super-individual 讲**个人杠杆**——本站讲**对外把东西 / 自己卖出去的实战体系**（给框架 + 真实话术 / 模板 / 案例，不停在道理层）。

> 顺序说明：A–D 是通用底层功（提问 / 价值 / 异议 / 成交 / 谈判）；学完先用到 **E 卖自己**（求职 / 融资 / 个人品牌，最普适也最有动力），再进 F 方法论体系与 G 规模化那套更专的 B2B 机器。

## Phase A · 底层认知（销售是什么）
- Day 1: 销售的本质 — 销售=帮对方做出对他更好的改变（价值交换，不是话术操纵）, 卖货/卖自己/卖想法是同一件事, 破除"我不是销售"的心魔 ｜cross-ref psychology 说服
- Day 2: 信任是唯一的货币 — 能力信任 vs 善意信任, 信任怎么建立与崩塌, 快速建立信任的具体动作, 长期信任复利
- Day 3: 买家的大脑 — 人为什么买 / 不买, 情绪决策 + 理性合理化, 损失厌恶与现状偏好, 风险与不确定性 ｜cross-ref psychology, mental-models
- Day 4: Jobs-to-be-Done — 人们"雇佣"产品完成什么任务, 功能 / 情感 / 社会三层需求, 真正的竞品是"不买"和替代方案
- Day 5: 卖结果不卖产品 — 特性 vs 好处 vs 结果, 客户要的是洞不是钻头, 把功能翻译成对方的成果

## Phase B · 搞清楚对方要什么（发现）
- Day 6: 会问才会卖 — 开放式提问, SPIN 提问法（背景 / 难点 / 暗示 / 需求-回报）, 问题漏斗 ｜Rackham《SPIN Selling》
- Day 7: 倾听的技术 — 主动倾听, 标注情绪（labeling）, 复述与确认, 沉默的力量 ｜Voss《Never Split the Difference》· cross-ref psychology
- Day 8: The Mom Test — 怎么问客户才不被善意谎言误导, 问过去的具体行为而非未来意向, 验证真需求 ｜Fitzpatrick
- Day 9: 挖痛点与放大 — 显性需求 vs 隐性痛点, 现状成本（不改变的代价）, 把痛点量化成钱 / 时间
- Day 10: 找到决策链 — 谁掏钱 / 谁拍板 / 谁使用 / 谁反对, B2B 的多决策人, 冠军（champion）与阻碍者 ｜MEDDIC
- Day 11: 资格判定（qualifying） — 不是每个客户都值得追, BANT / MEDDIC, 尽早说不省时间, 理想客户画像（ICP）

## Phase C · 讲清楚价值（叙事与说服）
- Day 12: 价值主张 — 一句话说清"你能帮谁解决什么、凭什么是你", 价值主张画布, 差异化 ｜cross-ref super-individual
- Day 13: 讲故事卖东西 — 故事结构（冲突-转折-结果）, 客户成功故事, 让对方成为故事主角 ｜Heath《Made to Stick》
- Day 14: 让信息黏住 — SUCCESs 原则（简单 / 意外 / 具体 / 可信 / 情感 / 故事）, 反直觉钩子, 用具体数字 ｜Heath
- Day 15: 影响力六原理 — 互惠 / 承诺一致 / 社会认同 / 权威 / 喜好 / 稀缺, 怎么用得不油腻, 抗操纵 ｜Cialdini《Influence》· cross-ref psychology
- Day 16: 做一个无法拒绝的报价 — 价值堆叠, 风险逆转 / 保证, 稀缺与紧迫, 锚定 ｜Hormozi《$100M Offers》
- Day 17: 演示与 pitch — 先痛后药, 少即是多, 现场 demo 的节奏, 让对方参与进来 ｜Klaff《Pitch Anything》
- Day 18: 电梯陈述与一页纸 — 30 秒讲清楚, 一句话定位, 电梯 pitch 的结构
- Day 19: 社会证明与案例 — 客户证言, 数据背书, logo 墙, 案例研究（case study）怎么写

## Phase D · 异议、谈判、成交
- Day 20: 异议不是拒绝 — 异议=信息与购买信号, 常见异议分类（不需要 / 不信 / 不急 / 太贵）, 先认同再回应
- Day 21: "太贵了"怎么办 — 价格 vs 价值, 拆分与对比, 不轻易打折的逻辑, 涨价的勇气
- Day 22: 谈判基础 — 立场 vs 利益, BATNA（最佳替代方案）, 做大蛋糕再分 ｜Fisher & Ury《Getting to Yes》· cross-ref leadership
- Day 23: 战术共情谈判 — 校准问题, 制造"说得对"的瞬间, 故意示弱, 黑天鹅信息 ｜Voss
- Day 24: 锚定与让步 — 谁先出价, 锚点怎么设, 让步的节奏与条件交换（不白让）
- Day 25: 成交的艺术 — 假设成交 / 二选一 / 紧迫成交, 识别购买信号, 敢于开口要, 沉默等待
- Day 26: 跟进的纪律 — 大量成交发生在第 5 次跟进之后, 有价值的跟进 vs 骚扰, 跟进节奏与话术
- Day 27: 处理"没回复"与死单 — 沉默背后的心理, 破冰话术, 复活死单, 优雅地放手

## Phase E · 卖自己（个人变现）
- Day 28: 卖自己是元技能 — 求职 / 融资 / 合作 / 影响力都是销售, 打磨个人价值主张, 克服自我推销的羞耻
- Day 29: 求职即销售 — 简历=落地页, 面试=发现 + demo, 谈 offer=谈判, 建立"被需要"感 ｜cross-ref super-individual
- Day 30: 融资 pitch — 讲清问题 / 市场 / 为什么是你, 投资人到底在买什么, pitch deck 结构, 处理质疑
- Day 31: 个人品牌 — 定位与差异化, 内容复利, 信任规模化, 从无名到被主动找上门 ｜cross-ref super-individual, writing
- Day 32: 自媒体变现 — 注意力 → 信任 → 变现漏斗, 产品阶梯设计, 社群, 不透支信任
- Day 33: 日常说服 — 让同事 / 伴侣 / 孩子接受你的想法, 非对抗式说服, 框架与时机 ｜cross-ref psychology, leadership
- Day 34: networking 不油腻 — 给予优先, 弱连接的价值, 长期关系账户, 怎么自然地开口求助 ｜cross-ref psychology

## Phase F · 方法论体系（顾问式 / Challenger / Sandler）
- Day 35: 顾问式销售 — 从卖方到顾问, 诊断-开方, 长期关系导向, 什么时候该"不卖"
- Day 36: 挑战者销售 — teach-tailor-take control, 用洞见挑战客户认知, 带来商业洞察 ｜Dixon & Adamson《The Challenger Sale》
- Day 37: Sandler 系统 — 前期契约, 痛苦漏斗, "不追不求"的姿态, 预算与决策前置
- Day 38: 价值销售（Value Selling） — 量化 ROI, 写商业案例（business case）, 与财务 / 高管对话
- Day 39: 解决方案销售的陷阱 — 过度定制, 复杂化, "解决方案疲劳", 何时回归简单直接

## Phase G · 渠道、漏斗与规模化（B2B / 增长）
- Day 40: 销售漏斗与管道 — 阶段划分, 转化率, 管道价值与预测, 漏斗健康度诊断
- Day 41: 找客户（prospecting） — 冷启动, 理想客户画像, 触达渠道选择, 外联的数字游戏 ｜Ross《Predictable Revenue》
- Day 42: 冷邮件与外联 — 主题行, 个性化, 价值前置, 序列与节奏, A/B 测试
- Day 43: 内容与 inbound — 让客户来找你, 教育式内容, 线索培育, SEO / 自媒体入口 ｜cross-ref writing
- Day 44: SaaS 与产品驱动增长 — PLG vs 销售驱动, 免费增值, 试用转化, 扩张收入（NRR）
- Day 45: 跨越鸿沟 — 早期采用者 vs 主流市场, 技术采用生命周期, 先打透一个细分 ｜Moore《Crossing the Chasm》
- Day 46: 定价的科学 — 价值定价 vs 成本加成, 套餐设计与价格锚定, 心理定价, 提价策略 ｜cross-ref investing, mathematics
- Day 47: 渠道与合作伙伴 — 直销 vs 分销 vs 代理, 联盟 / 分销, 平台与生态杠杆
- Day 48: 销售团队与流程 — 招 / 训 / 激励, SDR / AE / CSM 分工, CRM 与销售运营, 提成设计 ｜cross-ref leadership
- Day 49: 用数据做销售 — 关键指标（CAC / LTV / win rate / 周期）, 预测与复盘, 数据驱动改进 ｜cross-ref investing

## Phase H · 长期、伦理、进阶
- Day 50: 客户成功与复购 — 成交只是开始, onboarding, 减少流失, 向上销售 / 交叉销售
- Day 51: 口碑与转介绍 — 推荐飞轮, 什么时候要转介绍, NPS 与主动口碑
- Day 52: 销售的伦理边界 — 说服 vs 操纵, 暗黑模式, 长期主义, 卖不该卖的东西的代价 ｜cross-ref psychology 抗操纵
- Day 53: 心态与拒绝复原力 — 拒绝不针对你, 数字游戏心态, 抗挫与自我调节, 长期动机维持 ｜cross-ref psychology
- Day 54: 内向者的销售 — 深度倾听是优势, 一对一 vs 舞台, 真诚打法, 不必假装外向
- Day 55: 文化与跨文化销售 — 高 / 低语境, 关系导向 vs 交易导向, 本地化, 全球谈判差异
- Day 56: AI 时代的销售 — AI 做外联 / 线索 / 个性化, 什么被自动化 / 什么更值钱, 人的不可替代（信任 / 复杂谈判）, 用 AI 放大而非替代 ｜cross-ref ai-ml
- Day 57: 销售的神话与还原 — "天生会卖"的迷思, 话术 vs 体系, ABC（always be closing）的误读, 用数据打破直觉
- Day 58: 组合拳 · 一个完整成交复盘 — 从线索到成交到复购走一遍, 各阶段如何衔接, 一个端到端真实案例
